# Plans Folder Layout

- `plans/active/`: active plans to execute now.
- `plans/archive/YYYY-MM/`: completed plans archived by month.
- `plans/_template.plan.md`: required template.
- `plans/README.md` / `plans/PROMPTS.md`: process guidance.

## Archiving Rule

When all tasks in a `.plan.md` are `status: done`, move it to `plans/archive/YYYY-MM/`.
